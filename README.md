# Jogo do Número Secreto

Um jogo simples e estilizado onde o jogador tenta adivinhar um número secreto dentro de um intervalo definido.

## Status do projeto
🚧 Em desenvolvimento

---

## Objetivo do jogo

O jogador deve tentar adivinhar o número secreto gerado aleatoriamente pelo sistema.

A cada tentativa, o jogo pode informar se o número secreto é maior ou menor que o palpite do jogador.

---

## Funcionalidades atuais

- Escolha de um número pelo jogador
- Geração de número secreto aleatório
- Botão para chutar
- Botão para iniciar novo jogo
- Interface visual estilizada
- Validação básica de entrada
